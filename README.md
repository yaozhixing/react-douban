##仿豆瓣网电影网用 React 制作 webApp ###技术栈：React + react-route-dom + fetch

##演示地址： ####用的是豆瓣电影网公开 api 接口，地址是：https://douban.uieee.com， 文档：https://douban-api-docs.zce.me/movie.html#subject

##页面如下：

- 首页
- 正在热映
- 即将上映
- 北美票房榜
- 电影内容页
- 关于我
- 404

##有问题反馈

- react-router 管理路由
- 父子组件传递参数
- fetch 服务请求
- 星星评分系统，根据星星分数，处理样式 class 高亮状态
